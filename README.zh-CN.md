# ISAC DQN 资源分配项目简要说明

这是一个面向求职展示的公开版项目仓库，主题是使用深度强化学习研究 6G 集成感知与通信（Integrated Sensing and Communication, ISAC）场景下的资源分配问题。

## 项目做了什么

项目将一个通信侧的 Deep Q-Network（DQN）下行功率分配框架扩展到简化的 ISAC 仿真环境中。核心目标是观察智能体能否根据不同偏好，在通信吞吐量和感知性能之间形成可解释的权衡。

主要设计包括：

- 多小区下行链路仿真场景；
- 离散子带功率动作；
- 路径损耗、阴影衰落、瑞利衰落、天线增益、干扰、SINR、CQI 和吞吐量建模；
- 每个小区加入一个感知目标；
- 在状态和奖励中加入感知相关信息；
- 使用 `alpha` 控制通信与感知之间的偏好。

奖励函数的核心形式为：

```text
r_c = alpha * R_comm,c + (1 - alpha) * R_sense,c - lambda * R_power,c
```

其中 `alpha` 越大，策略越偏向通信吞吐量；`alpha` 越小，策略越偏向感知性能。

## 结果概览

实验比较了 DQN 与随机分配、固定功率、最大功率等简单基线，并对 `alpha = 0.0` 到 `1.0` 的 11 个偏好设置进行了评估。

从公开图表对应的结果看：

- 当 `alpha` 从 `0.0` 增加到 `1.0` 时，DQN 学到的平均感知功率比例从 `0.610` 降到 `0.182`；
- 平均吞吐量从 `132.39 Mbps` 增加到 `146.32 Mbps`；
- 结果体现了通信优先级提高后，策略会减少感知侧资源分配并提升通信性能；
- 鲁棒性图表展示了额外信道扰动下的吞吐量、感知分数和奖励变化。

主要图表可在 `assets/figures/` 中查看。

## 这个作品集展示的能力

- 将无线通信资源分配问题抽象成强化学习任务；
- 设计包含多目标权衡的奖励函数；
- 构建和解释通信-感知权衡实验；
- 使用基线和扰动实验辅助验证结果；
- 将毕业设计成果整理成适合公开展示的技术项目。
