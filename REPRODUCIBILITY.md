# Reproducibility Notes

This repository contains the runnable implementation, configuration files, final numerical CSV outputs, generated figures, and supporting documentation.

## Structure

```text
src/isac_dqn/
|-- environment.py
|-- radio.py
|-- dqn_agent.py
|-- baselines.py
|-- plotting.py
|-- replay_buffer.py
`-- utils.py
run_experiment.py
configs/config.yaml
results/final/
|-- summary_by_policy.csv
|-- evaluation_results.csv
|-- robustness_results.csv
|-- training_history.csv
`-- figures/
```

## Setup

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## Smoke Test

```powershell
python run_quick_test.py
```

The quick test writes outputs to `results/quick/` and model files to `models/quick/`, so it does not overwrite the included final result files.

## Full Runs

Run the configured full experiment:

```powershell
python run_experiment.py
```

Run the full experiment without GA baselines:

```powershell
python run_experiment.py --no-ga --output-dir results/full_no_ga --model-dir models/full_no_ga
```

## Result Checks

Check the included final CSV files and figures without requiring trained model files:

```powershell
python quality_check.py --results-dir results/final --skip-models
```

If regenerated model files are present, omit `--skip-models` to validate them as well.

## Included Outputs

The repository includes final CSV outputs and figures under `results/final/`.

## Excluded Outputs

Trained `.pt` model weight files are intentionally not committed. They can be regenerated by running the experiment scripts.
