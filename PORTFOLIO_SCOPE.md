# Public Portfolio Scope

This repository was created as a public job-portfolio version of a local dissertation project folder.

## Included

- Result figures from the local `figures/` folder.
- English project README for recruiter and hiring-manager review.
- Technical project summary.
- Reproducibility notes for future code publication.
- Scope notes explaining what was intentionally excluded.

## Excluded

The following local materials were intentionally not copied into this public repository:

- dissertation chapters and front matter,
- LaTeX source files,
- bibliography and build scripts,
- compiled dissertation PDFs,
- appendix PDFs,
- review notes,
- LaTeX build artifacts such as `.aux`, `.bcf`, `.log`, `.toc`, `.synctex.gz`,
- the source zip archive.

## Rationale

The goal is to show the project as engineering work without publishing private academic submission material, personal identifiers, or files that are not useful for a hiring review.

If the runnable project code is later located, it should be added in a separate commit with license checks, attribution, dependency files, and a reproducible experiment command.
